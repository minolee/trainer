accelerate, deepspeed, slurm 등을 사용하여 실행할 때 사용하는 config

command line에서 –XXX_config 를 통해 파일을 전달하면 해당 파일을 기반으로 XXX를 실행함 supported: python(default), accelerate, slurm

config file을 읽어서 해석하고 그에 맞는 command를 실행 python run.py –accelerate_config
